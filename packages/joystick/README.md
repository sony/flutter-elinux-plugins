# joystick

The implementation of the joystick plugin for eLinux.

## Usage

### pubspec.yaml
```yaml
dependencies:
  joystick:
    git:
      url: https://github.com/sony/flutter-elinux-plugins.git
      path: packages/joystick
      ref: main
```

### Source code
Import `joystick` in your Dart code:
```dart
import 'package:joystick/joystick.dart';
```
